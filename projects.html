Ongoing personal project to enhance practical skills in system administration, networking, and automation. RFID-Based Door Security System

Sep 2024 - Jan 2025

As part of my volunteer role within the IEEE Robotics and Automation Society (RAS) at UTAA, I led the development of an RFID-based door security system designed to enhance access control for our clubroom. The system is built using an Arduino Mega microcontroller and integrates RFID technology to allow secure, contactless entry for authorized members.

  • RFID Authentication, LCD feedback, user management
  • Expandable for network logging, keypad, biometrics
  • Technologies: Arduino Mega, RFID module RC522, LCD 16x2, C/C++ (Arduino IDE)
